# Plant-Disease-Detection
A CNN-based plant disease detection model built using MobileNetV2 Transfer Learning. Accuracy achieved was ~94% 

## Results
Result Details :- 
- Validation Accuracy: ~94%
- Classes: 15 (Pepper, Potato, Tomato — diseased and healthy)
- Dataset: Plant Village Dataset (~20,000 images)
 
Result Images Include :- 
- Training Curves
- Classification Report
- Confusion Matrix
- Sample Predictions
 
 ## Dataset
  PlantVillage Dataset available on Kaggle :- https://www.kaggle.com/datasets/adilmubashirchaudhry/plant-village-dataset
