# Plant-Disease-Detection
A CNN based model on "Plant Disease Detection" using MobileNetV2 Transfer Learning. Accuracy achieved was ~94% 

## Results
Result Details :- 
- Validation Accuracy: ~94%
- Classes: 15 (Pepper, Potato, Tomato — diseased and healthy)
- Dataset: Plant Village Dataset (~20,000 images)
 Dataset available on Kaggle :- https://www.kaggle.com/datasets/adilmubashirchaudhry/plant-village-dataset
 
Result Images Includes :- 
- Training Curves
- Classification Report
- Confusion Matrix
- Sample Predictions
